Graduation Project

In our project, we focus on the problem of Automated Personality Classification (APC). Firstly, we investigate possible improvements of the existing solutions to the problem of APC, for which we use different combinations of the APC linguistic analysis in text format for psychological trait measurements, and learning algorithms. We consider human resource recruitment in education fields to classify teachers according to their personality results according to (Five Factors Model of Personality Traits) using advanced data mining algorithm (Neural Network) by providing the algorithm with text inputs for our application (EduHR) as a simulation of interviews questions and possible answers.
